About the Quiz Web App

This web app lets you test your knowledge on various topics through quizzes. To access the quizzes, you must first create an account and log in. Each quiz is timed and includes multiple categories such as General Knowledge, Science, History, and more.

How to Get Started

  1. Register for a new account.
  2. Once registered, log in with your credentials.
  3. After logging in, you will be able to select quiz categories and start the quizzes.
